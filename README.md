# Burtsmon

A Cobblemon modpack for Minecraft 1.21.1 on NeoForge. Built around Cobblemon with performance optimizations, trainer battles, raid dens, mega evolutions, and quality-of-life additions.

## What's Included

**Cobblemon & Add-ons:**
- Cobblemon (core mod)
- Radical Cobblemon Trainers + RCTApi (NPC trainer battles)
- Cobblemon Raid Dens
- Cobblemon: Mega Showdown
- Cobblemon Pokenav
- Cobblemon Integrations
- Cobblemon Utility+ (IVs, EVs, & more)
- Cobblemon Size Variations
- Cobbreeding
- MoreCobblemonTweaks

**Performance:**
- Sodium, Lithium, FerriteCore, ImmediatelyFast, Entity Culling, AllTheLeaks

**Gameplay & Utility:**
- Sophisticated Backpacks & Storage
- Silent Gear
- JourneyMap
- Waystones
- JEI (Just Enough Items)

**Resource Packs:**
- AllTheMons R3.3
- Cobblemon Interface Recolored
- Maxi's JM+CM Pixel

**Datapacks:**
- AllTheMons R3.3

**Libraries (installed automatically):**
- Architectury API, GeckoLib, owo-lib, Accessories, Sophisticated Core, Silent Lib, Kotlin for Forge, Balm

## Installation

You'll need the [CurseForge App](https://www.curseforge.com/download/app) installed.

1. Download `burtsmon.zip` from this repo (click the file, then hit the download button on the right side).
2. Open the CurseForge app and make sure you're on the **Minecraft** section.
3. Click **Create Custom Profile** at the top.
4. Instead of filling anything out, click **Import** in the bottom-left corner of the profile creation screen.
5. Select the `burtsmon.zip` file you downloaded.
6. CurseForge will pull all the mods and apply the config overrides — give it a minute.
7. Once it's done, hit **Play**.

If CurseForge asks you about a mod loader, it should auto-detect NeoForge 21.1.220. If it doesn't, just make sure you have NeoForge selected with Minecraft 1.21.1.

## Troubleshooting

**Profile won't import:** Make sure you're running the latest version of the CurseForge app. Older versions sometimes choke on newer manifest formats.

**Mods failing to download:** A mod author may have removed or updated their file. Try clicking "Retry" in CurseForge, or check if there's an updated version of the pack in this repo.

**Low FPS:** The pack already includes Sodium, Lithium, and a few other performance mods, but if you're still struggling, try lowering render distance or turning off shaders if you have any installed separately.

**Out of memory / crashes on launch:** Bump your allocated RAM to at least 4GB. In CurseForge, go to the profile settings (three dots > Profile Options) and set the memory slider. 6-8GB is a comfortable spot if you have the RAM to spare.
